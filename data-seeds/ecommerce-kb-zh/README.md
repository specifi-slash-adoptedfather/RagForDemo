# 电商客服中文知识库种子

这是一份基于公开英文文档整理出来的中文客服知识库种子，目标是用于 RAG MVP 验证。

## 处理原则

- 不保留原始仓库中的工程噪音，只保留客服问答相关内容。
- 统一改写成中文客服语境，方便中文检索与回答。
- 每条 FAQ 或规则都附带来源映射，便于后续做引用展示。
- 当前内容以“整理摘要”为主，不是逐字翻译。

## 推荐导入顺序

1. `faq.jsonl`
2. `policies.md`
3. `sources.json`

## 文件说明

- `faq.jsonl`
  - 结构化问答对
  - 适合直接切 chunk 或逐条入库

- `policies.md`
  - 按主题整理的规则说明
  - 适合补充长文本上下文

- `sources.json`
  - 原始来源与主题映射
  - 便于前端展示“引用来自哪里”

## 注意

- 当前语料来自英文公开文档的中文整理版，适合 MVP 演示，不应直接作为正式业务规则。
- 如果后续要上线，应该替换为你自己的真实客服规则、售后政策、发票规则和物流 FAQ。
